## Usage Notes

- **Branded images** are the default choice for X — graphics render better in the feed than screenshots.
- **App screenshots** (especially hand shots) tend to outperform on Instagram and as TikTok thumbnails.
- **Rotate v1/v2 pairs** — never use both versions in the same post. Space them at least 2 weeks apart on the same angle.
- **Logo-only image (06)** should be used sparingly — max once per 10 posts. Reserve it for pure copy/quote posts.
- **Affiliate image (16)** is the only asset built exclusively for the capper/affiliate angle. Don't cross-use it with other pillars.
